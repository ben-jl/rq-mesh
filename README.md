# RQ-Mesh

## Set-up

1. Install Rust if necessary

    ```bash
    curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
    ```

2. Update Rust

    ```bash
    rustup update
    ```

3. Install gcc if necessary

    ```bash
    apk add gcc # Alpine Linux
    ```

4. Install sqlite3 if necessary

    ```bash
    apk add sqlite # Alpine Linux
    ```
